# Professional Resume Website

A detailed Resume of Oluwanisola Dele-Adelusi Joshua showing his qualifications, skills and journey so far.

## Features

- **Minimalist Design**: Clean, table-style layout inspired by seanoshea.me
- **Responsive Design**: Works on all devices with mobile-first approach
- **Dark/Light Mode**: Toggle between color schemes for comfortable viewing
- **Semantic HTML Structure**: Proper use of HTML5 elements for better accessibility
- **Professional Content**: Comprehensive presentation of education, experience, and skills
- **Easy Navigation**: Clear links between sections and pages

## Files

- `index.html` - Main resume page with education, experience, skills, and awards
- `contact.html` - Contact information and form
- `minimal-style.css` - Stylesheet with minimal aesthetic
- `main.js` - JavaScript for theme toggle and form handling
- `assets/images/` - Profile image directory
